# 🛍️ 가격비교 - 최저가를 한눈에

한국 주요 쇼핑몰의 상품 가격과 모든 혜택을 실시간으로 비교하는 프리미엄 웹 애플리케이션입니다.

## ✨ 주요 기능

### 💰 포괄적인 혜택가 계산
- **쿠폰 할인**: 각 쇼핑몰의 할인 쿠폰 적용
- **카드 할인**: 제휴 카드사 즉시 할인 혜택
- **멤버십 할인**: 로켓와우, VIP 회원 등 멤버십 추가 할인
- **포인트 적립**: 네이버페이, 로켓와우, OK캐시백 등 적립 혜택
- **무료배송**: 배송비 무료 혜택

### 🔍 스마트 검색
- 실시간 상품 검색
- 인기 검색어 원클릭 검색
- 4개 주요 쇼핑몰 비교 (네이버쇼핑, 쿠팡, 11번가, G마켓)

### 📊 다양한 정렬 옵션
- **전체**: 기본 정렬
- **최저가순**: 최종 혜택가 기준 정렬
- **할인율순**: 할인율이 높은 순서
- **혜택순**: 총 혜택 금액이 큰 순서

### 🎨 프리미엄 디자인
- 다크 테마 기반 모던한 UI
- 부드러운 애니메이션과 트랜지션
- 반응형 디자인 (모바일/태블릿/데스크톱)
- 그리드/리스트 뷰 전환

## 🚀 사용 방법

### 1. 파일 열기
브라우저에서 `index.html` 파일을 열어주세요.

### 2. 상품 검색
- 검색창에 상품명을 입력하거나
- 인기 검색어 태그를 클릭하세요

### 3. 결과 확인
- 각 쇼핑몰의 가격과 혜택을 비교
- 최저가 상품은 파란색 테두리와 "최저가" 배지로 표시
- 총 혜택 금액과 최종 혜택가 확인
- 적립 포인트 별도 표시

### 4. 정렬 및 필터링
- 상단 필터 버튼으로 정렬 방식 변경
- 그리드/리스트 뷰 전환

```
├── index.html      # 메인 HTML 파일

├── script.js       # JavaScript 로직 (검색, 필터링, 혜택 계산)
└── README.md       # 프로젝트 설명서
```

## 🎯 혜택가 계산 방식

### 최종 혜택가 계산
```
최종 혜택가 = 판매가 - (쿠폰 할인 + 카드 할인 + 멤버십 할인)
```

### 총 혜택
```
총 혜택 = 쿠폰 할인 + 카드 할인 + 멤버십 할인
```

### 적립 포인트
포인트는 할인이 아닌 적립 혜택이므로 최종 혜택가 계산에는 포함되지 않고 별도로 표시됩니다.
