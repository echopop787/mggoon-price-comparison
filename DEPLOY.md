# 맛꾼푸드 가격비교

한국 주요 쇼핑몰의 상품 가격과 모든 혜택을 실시간으로 비교하는 프리미엄 웹 애플리케이션

## 배포 방법

### Vercel로 배포하기

1. **Vercel CLI 설치** (처음 한 번만)
```bash
npm install -g vercel
```

2. **로그인**
```bash
vercel login
```

3. **배포**
```bash
cd "d:\새 폴더 (2)"
vercel
```

4. **프로덕션 배포**
```bash
vercel --prod
```

### 또는 Vercel 웹사이트에서 배포

1. https://vercel.com 접속
2. GitHub/GitLab/Bitbucket 연동 또는 파일 직접 업로드
3. 프로젝트 폴더 선택
4. 자동 배포 완료!

## 배포 후 사용

배포가 완료되면 `https://your-project.vercel.app` 형태의 URL을 받게 됩니다.
이 URL을 MD나 팀원들과 공유하여 어디서든 가격비교를 할 수 있습니다.

## 주요 기능

- ✅ 13개 플랫폼 가격 비교
- ✅ 실시간 가격 업데이트
- ✅ 최종 혜택가 계산
- ✅ 맛꾼푸드 가격 경쟁력 분석
